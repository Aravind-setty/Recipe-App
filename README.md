
# Recipe App

Recipe App is a simple web application used to search Recipe by Name...


## Run this App Locally

Clone the project

```bash
  git clone https://github.com/Aravind-setty/Recipe-App 
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
